# PRD: AX 코칭단 평가 시스템
## Product Requirements Document

---

## 1. 개요

### 1.1 프로젝트 비전
스마트팩토리 AX(AI Transformation) 전환을 위한 기업-전문가 매칭 및 코칭 관리 시스템

### 1.2 문제 정의
- 기업 방문 AX 코칭 과정의 체계적 관리 부재
- 전문가-기업 간 효율적인 매칭 시스템 필요
- 코칭 결과 및 품질 평가의 표준화 필요

### 1.3 목표
1. **기업-전문가 매칭 효율화**: AI 기반 최적 매칭 추천
2. **코칭 품질 평가**: 체계적인 평가 및 점수 관리
3. **AX 방향 점검 관리**: 체크리스트 기반 점검 시스템

### 1.4 성공 지표
- 코칭 보고서의 체계적 관리 달성
- 매칭 성공률 80% 이상
- 평가 완료율 95% 이상

---

## 2. 사용자 정의

### 2.1 주요 사용자
| 역할 | 설명 | 주요 기능 |
|------|------|----------|
| ADMIN (전문가) | 시스템 관리 및 코칭 전문가 | 전체 기능, 매칭 관리, 보고서 조회 |
| EVALUATOR | 평가 담당자 | 평가 채점, 점수 관리 |
| APPLICANT | 지원자 (예비 전문가) | 평가 응시, 결과 확인 |

### 2.2 운영 주체
- 민간 컨설팅 조직
- 코칭단 코칭 위원

---

## 3. 핵심 기능 요구사항

### 3.1 기업 AX 점검 체크리스트 작성
```
우선순위: P0 (필수)
상태: 구현 완료
```

**기능 상세:**
- 기업별 AX 성숙도 점검 항목 관리
- 분야별 체크리스트 (ML, DL, CV, 데이터 인텔리전스 등)
- 난이도별 문항 구성 (EASY/MEDIUM/HARD)
- 객관식/주관식/서술형 문항 지원

**수용 기준:**
- [ ] 카테고리별 문항 생성 가능
- [ ] 배점 및 가중치 설정 가능
- [ ] 전문분야별 필터링 지원

### 3.2 코칭 보고서 작성 및 제출
```
우선순위: P0 (필수)
상태: 구현 완료
```

**기능 상세:**
- 전문가별 평가 결과 보고서 생성
- PDF 다운로드 지원 (한글 폰트 지원)
- 시스템 전체 통계 보고서
- 매칭 분석 보고서

**수용 기준:**
- [ ] 개별 전문가 평가 보고서 생성
- [ ] 시스템 요약 보고서 생성
- [ ] PDF 다운로드 기능

### 3.3 기업 매칭 현황 확인
```
우선순위: P0 (필수)
상태: 구현 완료
```

**기능 상세:**
- AI 기반 전문가 추천 (가중치 기반 점수)
  - 전문분야 일치도 40%
  - 평가 성과 20%
  - 자격 검증 15%
  - 경력 15%
  - 가용성 10%
- 매칭 현황 대시보드
- 매칭 이력 관리

**수용 기준:**
- [ ] 수요에 대한 TOP N 전문가 추천
- [ ] 호환성 점수 시각화
- [ ] 매칭 상태 추적 (제안→수락→진행→완료)

## 5. 비기능 요구사항

### 5.1 성능
- API 응답 시간: 200ms 이하 (95 percentile)
- 동시 사용자: 100명 이상 지원
- 매칭 추천 시간: 2초 이내

### 5.2 보안
- JWT 기반 인증
- 역할 기반 접근 제어 (RBAC)
- 비밀번호 해시화 (bcrypt)
- HTTPS 필수

### 5.3 가용성
- 시스템 가용성: 99.5%
- 정기 백업: 일일 1회

## 6. 제약사항

### 6.1 기술적 제약
- Python 3.11+ 필수
- PostgreSQL 14+ 필수
- Node.js 18+ 필수 (프론트엔드)

### 6.2 운영적 제약
- 한국어 UI 필수
- 한글 PDF 출력 지원
