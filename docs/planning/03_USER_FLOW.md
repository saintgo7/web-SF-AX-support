```

│                        로그인 페이지                          │
```

## 2. ADMIN (관리자/전문가) 플로우

### 2.1 메인 네비게이션
```
/admin
├── /dashboard     - 대시보드 (통계 요약)
├── /experts       - 전문가 관리
├── /questions     - 평가 문항 관리
├── /matching      - 기업 매칭 관리
└── /reports       - 보고서 관리
```

## 5. 공통 플로우

### 5.1 인증 플로우
```
┌─────────────┐     ┌─────────────┐     ┌─────────────┐
│   로그인     │────▶│  JWT 발급   │────▶│  대시보드    │
│  /auth/login│     │ (Access+    │     │ (역할별)    │
│             │     │  Refresh)   │     │             │
└─────────────┘     └─────────────┘     └─────────────┘
                          │
                          ▼ 토큰 만료
              ┌─────────────────────┐
              │    자동 토큰 갱신     │
              │  (Refresh Token)    │
              └─────────────────────┘
```

### 5.2 에러 처리 플로우
```
API 호출 실패
      │
      ├─ 401 Unauthorized ──▶ 로그인 페이지로 이동
      │
      ├─ 403 Forbidden ────▶ "권한이 없습니다" 알림
      │
      ├─ 404 Not Found ────▶ "데이터를 찾을 수 없습니다" 알림
      │
      └─ 500 Server Error ─▶ "서버 오류가 발생했습니다" 알림
```
